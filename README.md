# To-Do-List Uygulması 
Bu proje HTML, CSS ve JavaScripy kullanılarak geliştirilmiş basit bir yapılacaklar listesi uygulaasıdır.

##Özellikler
-Görev Ekleme
-Görev Silme
-Görev tamalandı olarak işaretleme 
-Basit arayüz

##Kulanılan Arayüzler
-HTML
-CSS
-JavaScript

## Amaç
Bu proje, JavaScript yapısını öğrenmek ve temel fronted becerilerini geliştirmek için yapılmıştır.

## Not
Bu proje öğrenme amaçlıdır geliştirilecektir.
